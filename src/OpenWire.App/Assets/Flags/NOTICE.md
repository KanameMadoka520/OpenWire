# Country flag icons

The 16×11 PNG country flags in this folder are the **famfamfam "Flag" icon set**
by Mark James (http://www.famfamfam.com/lab/icons/flags/).

> These icons are public domain, and as such are free for any use (attribution
> appreciated but not required).

They are named using ISO 3166-1 alpha-2 country codes, which lets OpenWire map a
GeoIP country code straight to a flag with no lookup table.

Bundled here (rather than fetched at runtime) so the Traffic / Usage screens work
fully offline. Not affiliated with GlassWire or any other product.
